# Dashboard Comercial IAF 2026

## Instalação (Windows) — executar uma única vez

### Passo 1 — Instalar o Python
1. Acesse https://www.python.org/downloads/
2. Clique em "Download Python 3.x"
3. Execute o instalador
4. **OBRIGATÓRIO:** marque "Add Python to PATH" antes de clicar em Install

### Passo 2 — Instalar o Dashboard
1. Extraia esta pasta em qualquer lugar (ex: `C:\Dashboard`)
2. Dê duplo clique em **instalar.bat**
3. Aguarde (2-3 minutos)

---

## Uso diário
1. Dê duplo clique em **iniciar.bat**
2. O navegador abre em http://localhost:8501
3. Faça login: usuário `admin`, senha `admin2026`
4. Carregue os arquivos CSV na barra lateral

---

## Corrigir um cálculo errado
Todos os cálculos estão em `modulos/calculos.py`.
Edite a função correspondente e reinicie com `iniciar.bat`.
